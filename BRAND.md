# VANTA Brand Guide

## Identity

**Name:** VANTA
**Tagline:** Trade smarter. Trade faster.
**Voice:** Confident, direct, modern. Not stuffy or "Wall Street." Not childish either.

## Logo Concept

Stylized **V** with an upward chart line integrated into the right stroke. The V doubles as an arrow. Strong, geometric, works at small sizes (favicon, app icon).

```
   /\        /\
  /  \      /  \
 /    \    /    \
/______\  /______\
              ▲
              │ price
              │
              ▼
```

App icon: solid black square (#0A0E1A) with the V mark in electric blue.

## Color Palette

| Role | Hex | Use |
|---|---|---|
| **Primary** | `#3B82F6` | Buttons, links, brand accents |
| **Primary Glow** | `#60A5FA` | Hover/active states |
| **Profit Green** | `#10D984` | Wins, gains, positive P&L |
| **Loss Red** | `#FF3B5C` | Losses, sells, negative P&L |
| **BG Deep** | `#0A0E1A` | App background |
| **BG Elevated** | `#131829` | Cards, modals |
| **BG Surface** | `#1C2236` | Inputs, secondary surfaces |
| **Border** | `#2A3148` | Dividers, card outlines |
| **Text Primary** | `#FFFFFF` | Body text |
| **Text Secondary** | `#8B92A6` | Labels, hints |
| **Text Muted** | `#5A6178` | Disabled, timestamps |

Default theme is **dark**. Light mode is a future addition.

## Typography

- **Display:** Inter (700/600) — headers, titles
- **Body:** Inter (400/500) — paragraphs, UI text
- **Numbers:** JetBrains Mono — prices, P&L, account balances (tabular figures essential)

## Tone Examples

| Context | Voice |
|---|---|
| Welcome | "Welcome to Vanta. Let's get trading." |
| Trade win | "+$152.30 — nice." |
| Trade loss | "−$48.20 — that one didn't go your way." |
| Demo banner | "You're in demo. No real money is at risk." |
| Empty state | "No positions yet. Hit Trade to open one." |
| Error | "Something went wrong on our end. Try again?" |

Avoid:
- "Congratulations on your trade!" (too formal)
- "OMG you won big!" (too casual)
- Emojis in serious financial contexts

## Mode Switcher

Two modes, switchable from the profile header:

- **Pro Mode** — chart-first, MT4-style
- **Quick Mode** *(internal: "Fun")* — binary-style, gamified

Naming "Fun" externally as **Quick Mode** sounds more legitimate while keeping the same simplified UX.
